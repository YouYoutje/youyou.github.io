### lib.disableControls

Call on frame to disable all stored keys.

```lua
lib.disableControls()
```

### lib.disableControls:Add

```lua
lib.disableControls:Add(...)
```

* vararg: `number`

```lua
lib.disableControls:Remove(...)
```

* vararg: `number`

### lib.disableControls:Clear

```lua
lib.disableControls:Clear(...)
```

* vararg: `number`
