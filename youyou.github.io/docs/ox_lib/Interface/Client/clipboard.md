### lib.setClipboard

Sets the player's clipboard to the specified string value.

<Tabs>
<TabItem value='Lua'>

```lua
lib.setClipboard(value)
```

</TabItem>
<TabItem value='JS/TS'>

```ts
import lib from '@overextended/ox_lib/client'

lib.setClipboard(value)
```

</TabItem>
</Tabs>

* value: `string`
